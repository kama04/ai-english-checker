# AI English Checker

Simple website for checking English spelling and grammar with OpenAI.

## 1. Install
```bash
npm install
```

## 2. Add your API key
Create a `.env` file and paste:

```env
OPENAI_API_KEY=your_openai_api_key_here
PORT=3000
```

## 3. Run
```bash
npm start
```

Open in browser:
```bash
http://localhost:3000
```

## Project structure
- `public/index.html` — frontend
- `server.js` — backend with OpenAI API
- `.env.example` — example env file
